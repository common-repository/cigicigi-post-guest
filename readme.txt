== Description ==

Guest Post easyl post new for the guest. This posts do not public. Only admin public posts. This plugin include standart text editor and BBCode editor, also include simple captcha and reCAPTCHA. Guests only one upload image or do not upload image option.

== Installation ==

1. Upload `post-guest-cigicigi` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in your WordPress admin panel.
3. Create a page or a post and insert shortcode [post_guest_cigicigi] to the text.
